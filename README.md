# KaryaPro | Professional Portfolio "OP" Edition

KaryaPro adalah website portofolio profesional kelas atas yang dibangun dengan teknologi modern dan fitur kecerdasan buatan (AI).

## 🚀 Fitur Utama
- **Real-time Database:** Menggunakan Firebase Firestore untuk sinkronisasi data instan.
- **AI-Powered:** Integrasi Genkit AI untuk terjemahan otomatis (ID ↔ EN) dan saran deskripsi proyek.
- **Smart Navigation:** Navbar dan Admin Tabs dengan sistem *auto-hide* saat scroll.
- **Multi-Language:** Dukungan penuh untuk Bahasa Indonesia dan Inggris.
- **Multi-Theme:** Tersedia berbagai tema (Light, Dark, Cyber, Emerald, dll.).
- **Dynamic Social Connectivity:** Integrasi WhatsApp, LinkedIn, GitHub, Instagram, dan TikTok.

## 🎨 Desain & Gaya
- **Typography:** Tipografi raksasa menggunakan font *Space Grotesk*.
- **Visuals:** Efek Aura Background, Card Glow, dan Glassmorphism 2.0.
- **UI/UX:** Animasi transisi halus dan responsif di semua perangkat (Mobile-first).

## 🛠 Tech Stack
- **Frontend:** Next.js 15 (App Router), React 19, TypeScript.
- **Styling:** Tailwind CSS, ShadCN UI, Lucide Icons.
- **Backend:** Firebase Firestore (Database) & Firebase Authentication.
- **GenAI:** Google Gemini via Genkit AI.

## 🔐 Akses Admin
Halaman Admin dapat diakses melalui `/admin/login`. 
*Catatan: Login menggunakan akun yang terdaftar di Firebase Authentication proyek ini.*
